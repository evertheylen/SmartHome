To make a group, you'll have to press the matching button here: Give your group an awesome name and you're ready to rock!
You can join a group by pressing the button below the button to make a group. You'll then get a form where you can select the group you want to join.
You can find friends by pressing the matching button here: You'll get a list of people, and you'll be able to search for people using the search bar. You add them by clicking the "add friend" button.
The profile of a friend: navigate to "My Friends" using the menu drawer (see How do i navigate?) and click on the name of the friend you want to inspect.
